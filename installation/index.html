Installing The Server Xampp

  1. Open the CD disk find the Xampp application and install it (Do not change the installation directory, it will automatically installed in C Drive)
  2. When comfirmation prompts appear make sure to click "Yes".
  3. A Xampp Setup window will appear, just click next, next ..... until installation starts.
  4. After finish the installation go to C:\xampp and find xampp control-panel application and open it.
  5. A window like this will open, start the two services Apache and MySQL.

Copying the software files to xampp folder

After installing xampp and starting the two services(Apache and MySQL). now lets copy the software files in the xampp folder.

  1. Go to the CD Directory
  2. Copy the folder named "poslite" to C:\xampp\htdocs

Software Database Setup

Setting the software database is simple just click this link after the setup it will automatically redirect you to the login page and you can now start using the software.

Login Credentials

The default admin username is admin and password is admin123 (Please do not delete the admin user, you can add new user in the system).
